## 프로젝트 배경

```text
1) 문제점
- 33년 전통 오프라인 매장의 강점과 분위기를 전달할 공식 웹 채널이 부족했습니다.
- 메뉴, 위치, 영업시간, 주차, 예약 문의 정보가 온라인에서 한 번에 정리되어 있지 않았습니다.
- 외국인 방문객이나 타지역 고객이 메뉴와 매장 정보를 이해하기 어려웠습니다.

2) 프로젝트 목표
- 노포의 따뜻함과 신뢰감을 담은 공식 브랜드 웹사이트를 구축합니다.
- 한국어, 영어, 중국어 3개 국어로 메뉴와 매장 정보를 제공합니다.
- 메뉴 확인 → 위치 확인 → 전화 예약으로 이어지는 방문 전환 흐름을 단순화합니다.

3) 주안점
- 템플릿 느낌을 줄이고 실제 매장 분위기가 느껴지는 디자인
- 모바일 우선 반응형 구조와 직관적인 CTA
- 검색엔진 수집과 지도앱 이동까지 고려한 운영 가능한 웹사이트
```

## 프로젝트 상세

```text
1) 프로젝트 소개
- 제천의 33년 전통 갈비 전문점 '새영동숯불갈비'를 위한 공식 브랜드 웹사이트 외주 프로젝트입니다.
- 단순 소개형 홈페이지가 아니라, 메뉴 확인부터 위치 확인, 전화 문의까지 실제 방문 전환 흐름을 자연스럽게 연결하는 데 초점을 맞췄습니다.

2) 담당 범위
- 서비스 구조 기획, UI 톤앤매너 정리, 프론트엔드 개발, 다국어 적용, 지도/예약 CTA 구성, SEO 세팅, 배포까지 전 과정을 담당했습니다.
- 전통 식당의 분위기를 온라인에서도 신뢰감 있게 전달할 수 있도록 디자인 방향과 정보 구조를 함께 설계했습니다.

3) 주요 구현 내용
- 메인 페이지에는 브랜드 스토리, 직접 발골, 직접 뽑은 함흥냉면, 한약재 양념 등 매장의 핵심 강점을 중심으로 콘텐츠를 구성했습니다.
- 메뉴 페이지는 고기·식사·주류 3개 카테고리, 총 21개 메뉴를 가격과 설명 중심으로 구조화해 탐색성을 높였습니다.
- 방문 전환을 위해 주소, 영업시간, 주차 정보, 전화 연결, 네이버/카카오맵 이동 버튼을 한 섹션에 통합했습니다.
- 단체 방문 수요를 고려해 프라이빗 룸 갤러리와 매장 공간 정보를 함께 제공했습니다.

4) 기술 포인트
- Next.js App Router와 TypeScript 기반으로 반응형 웹사이트를 구현했습니다.
- next-intl을 적용해 한국어·영어·중국어 3개 국어 라우팅과 번역 데이터를 분리했습니다.
- 언어 전환 시 현재 경로를 유지하도록 설계해 사용 흐름이 끊기지 않게 했고, 카카오맵 SDK는 로딩 시점을 제어해 초기화 안정성을 확보했습니다.
- 메타데이터, Open Graph, JSON-LD, sitemap, robots, hreflang을 적용해 검색엔진 노출과 공유 대응 기반도 함께 구축했습니다.

5) 프로젝트 의의
- 전통 식당의 브랜드 경험을 템플릿형 소개 페이지가 아니라 실제 방문과 예약 전환 중심의 웹 경험으로 정리한 프로젝트입니다.
- 디자인적으로는 warm beige 톤, serif 타이포그래피, organic shadow를 활용해 템플릿스럽지 않은 '깨끗한 예스러움'을 구현하는 데 집중했습니다.
```
