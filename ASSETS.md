# 필요한 이미지 에셋 리스트 (Required Assets)

현재 웹사이트는 AI로 생성된 기본 이미지와 플레이스홀더를 사용하고 있습니다. 브랜드의 신뢰도를 높이기 위해 아래의 실제 고화질 사진들이 필요합니다.

## 1. 메인 섹션 (Hero Section)
*   **파일 이름 (권장)**: `hero-main.jpg`
*   **내용**: 식당의 대표 메뉴(한우 생갈비)와 정갈한 상차림이 돋보이는 고화질 사진.
*   **권장 사이즈**: 1920x1080px 이상 (가로형)

## 2. 브랜드 스토리 (Story Section)
*   **파일 이름**: `story-kitchen.jpg`, `story-hall.jpg`
*   **내용**:
    *   `kitchen`: 사장님이 직접 고기를 손질(발골)하는 역동적인 모습.
    *   `hall`: 깔끔하고 전통적인 매장 내부 전경 또는 단체실 모습.
*   **권장 사이즈**: 800x1200px (세로형 또는 정사각형)

## 3. 핵심 강점 (Features Section)
*   **파일 이름**: `feature-naengmyeon.jpg`, `feature-butchering.jpg`
*   **내용**:
    *   `naengmyeon`: 100% 고구마 전분으로 직접 면을 뽑는 과정 또는 완성된 함흥냉면 근접샷.
    *   `butchering`: 최상급 한우 생고기의 마블링이 잘 표현된 디테일 샷.
*   **권장 사이즈**: 1000x1000px (정사각형)

## 4. 갤러리 (Gallery Section)
*   **파일 이름**: `room-1.jpg` ~ `room-10.jpg`
*   **내용**: 각 개별 룸(1번방~10번방)의 내부 사진. 가족 모임이나 회식 장소로서의 쾌적함을 강조.
*   **권장 사이즈**: 1200x800px (가로형)

---

### 💡 이미지 최적화 팁
1.  **파일 형식**: 가능하면 `.webp` 형식을 권장하지만, 원본(`.jpg`, `.png`)을 주시면 제가 직접 최적화하여 반영해 드릴 수 있습니다.
2.  **용량**: 웹사이트 로딩 속도를 위해 파일당 1MB 이하로 압축하는 것이 좋습니다.
3.  **반영 방법**: `public/images/` 폴더에 위 이름으로 저장하면 자동으로 웹사이트에 반영되도록 코드를 수정해 드릴 예정입니다.
